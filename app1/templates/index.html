{{ fortune }}

This is a basic python flask web application which is hosted on an AWS EC2 instance, in an autoscaling group, load balanced across multiple availability zones. This application was created to demonstrate my abilities with different AWS technologies. The application retrieves data from a basic aws dynamo database upon each load, and displays it on the page. This application was created from aws modules that are entirely managed using terraform configuration files. This includes everything from the VPC, to instance, to DNS configurations and more.
